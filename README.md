# 영화리뷰 품질 분석기


### 프로젝트 추진배경
데이터의 폭발적인 증가로 데이터를 가공하여 사용하고자 하는 노력이 많아지고 있습니다. 이에 데이터를 가공하여 오류단어 및 deep learning기반으로 품질을 제시해주고자 하였습니다.

### 기능
- 뉴스 기사 댓글, 영화 리뷰 크롤링
- 오류 사전 기반 데이터 품질 분석
- 의사결정 트리 기반 데이터 품질 분석
- 품질 분석 결과 도표화

### 프로젝트 수행 기간
 2020.05. ~ 2020.11

### 알고리즘
- random Forest
- Rnn
